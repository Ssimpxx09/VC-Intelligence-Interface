# VC Intelligence Interface

## Features
- Company discovery dashboard
- Search, filters, sorting, pagination
- Company profile with live enrichment
- Notes + Save to list
- Lists page + Export
- Saved searches
- Real website enrichment via server API

## Tech
- Next.js
- LocalStorage persistence
- Server-side enrichment API

## Setup

Install dependencies:
npm install

Run locally:
npm run dev

## Environment Variables

Create `.env.local`:

OPENAI_API_KEY=your_key
